Bit-Flip & Phase-Flip Repetition Codes

1. Introduction

Quantum computers are susceptible to noise, which introduces errors in computation. Quantum Error Correction (QEC) is the field of protecting quantum information from these errors. This project explores the simplest form of QEC: the Repetition Code.

Project Goal: Simulate quantum circuits using Stim to demonstrate how encoding one logical qubit into multiple physical qubits can significantly reduce error rates.

2. Methodology

We implemented two types of repetition codes to handle different noise models:

2.1 Bit-Flip Code (Z-Basis)

Protects against Bit-Flip (X) errors ($X|0\rangle = |1\rangle$).
Encoding: $|0\rangle_L = |000\rangle$, $|1\rangle_L = |111\rangle$.
Syndrome Check: Measures Z-parity ($Z_i Z_{i+1}$) to detect flips without collapsing the state.

2.2 Phase-Flip Code (X-Basis)

Protects against Phase-Flip (Z) errors ($Z|+\rangle = |-\rangle$).
Encoding: $|0\rangle_L = |+++\rangle$, $|1\rangle_L = |---\rangle$.
Syndrome Check: Measures X-parity ($X_i X_{i+1}$).

4. Conclusion

This project successfully demonstrates the principles of quantum error correction. We observed that:

  • Repetition codes can arbitrarily reduce logical error rates given low enough physical noise.
  • There is a distinct difference between correcting X-errors and Z-errors, requiring basis transformations (Hadamard gates).
  • Measurement errors significantly impact the overall fidelity, introducing a trade-off for larger codes.
